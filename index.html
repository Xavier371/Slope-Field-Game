Slope Field Game

Enter an equation for y' in terms of x and y
to map the red point through the yellow boundaries.

How it works

Input y' = f(x, y). The slope of each arrow is the value of y' at the given (x,y) coordinate. The red curve is a particular solution of the equation given an initial point.

Your goal: write a function so that the curve exits through both highlighted yellow boundaries. Tap Reset for a new initial point, and boundary conditions.
