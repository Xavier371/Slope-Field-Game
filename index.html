How to Play

What is a slope field?

A slope field visualizes \(y' = f(x,\, y)\), where \(f(x,\, y)\) is the slope of each arrow at the corresponding point \((x,\, y)\).

Your goal

Select \(f(x,\, y)\) so that, starting from the red point, the solution curve intersects both highlighted boundary segments.

Supported syntax

Variables \(x\) and \(y\)  ·  operators \(+,\,-,\,*,\,/,\,\hat{\phantom{i}}\)  ·  functions \(\sin,\,\cos,\,\tan,\,\exp,\,\ln,\,\sqrt{\cdot}\), etc.

Try: \(x \;\cdot\; {-y} \;\cdot\; x+y \;\cdot\; x^2-y \;\cdot\; \sin(x)\cos(y)\)

Solve button

Press Solve and the game finds a working \(f(x,\, y)\) for you.
